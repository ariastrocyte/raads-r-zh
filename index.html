RAADS-R 是什么？

RAADS-R（Ritvo Autism Asperger Diagnostic Scale–Revised） 是一份用于帮助识别成年孤独症谱系障碍（ASD）个体的临床辅助工具，由 Riva Ariella Ritvo 博士 等人开发，并于 2010 年扩大样本进行验证研究。

该量表共有 80 道关于社交、生活体验和习惯的单选题，每道题有四个选项：

  • True now and when I was young（现在和小时候都符合）
  • True now only（仅现在符合）
  • True only when I was younger than 16（仅16岁之前符合）
  • Never true（从不符合）
